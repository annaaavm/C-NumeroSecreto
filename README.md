# Jogo do Número Secreto

Projeto simples desenvolvido em C com o objetivo de praticar lógica de programação, estruturas condicionais e laços de repetição.

## Como funciona
O programa gera um número aleatório e o usuário deve tentar adivinhar qual é o número secreto.

## Tecnologias
- Linguagem C

## Contexto
Projeto desenvolvido durante estudos a partir de curso da plataforma Alura.
